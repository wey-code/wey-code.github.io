# MD常用语法格式

## 一、标题

  一般使用#来标识层级。一共六个层级，再多不再标识。

   \# => h1;

  \## => h2;

  \### => h3;

  \#### => h4;

  \##### => h5;

  \###### => h6;
  
  注：h1 级别会默认带一个 \<hr/>

## 二、分割线
  至少三个 * 或 - 或 _ ，有空格也可以，不必连续。
  
## 三、换行
  两个空格 + 一个回车。
